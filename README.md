# Boletim Universitário Semestral

Projeto desenvolvido em Python para simular um sistema de boletim universitário.

## Funcionalidades

- Cadastro de múltiplos alunos
- Validação de notas e frequência
- Cálculo da média semestral
- Verificação de aprovação, recuperação ou reprovação
- Recuperação com nova avaliação
- Geração de boletim individual
- Relatório final da turma
- Estatísticas de aprovação e reprovação

## Tecnologias

- Python 3

## Objetivo

Aplicar conceitos de:
- Estruturas condicionais
- Laços de repetição
- Funções
- Validação de dados
- Manipulação de listas
